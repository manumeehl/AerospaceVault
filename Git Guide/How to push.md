after completing editing a file, type this in the terminal in the repository folder:


1: "git add <file>" -> 

2.1: "git commit <file>"
2.2: (enter commit message: what changed)
2.3: "[esc]"
2.4: ":wq"

3: "git push" -> pushes you changes to the repository



Abort Commit:
2.2: (leave commit message empty)
2.3: "[esc]"
2.4: ":q"


:q -> quit
:wq -> write and quit
